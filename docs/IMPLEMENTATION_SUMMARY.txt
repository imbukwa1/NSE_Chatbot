DATABASE STRUCTURE
==================

nse_stocks.db:
├─ stocks table
│  ├─ ticker (TEXT, PRIMARY KEY + UNIQUE INDEX)
│  ├─ name (TEXT)
│  ├─ price (REAL)
│  ├─ change_pct (REAL)
│  ├─ volume (INTEGER)
│  ├─ pe_ratio (REAL)
│  ├─ dividend_yield (REAL)
│  ├─ source (TEXT: "live_scrape" or "seed")
│  └─ updated_at (TIMESTAMP, auto-updated on INSERT OR UPDATE)
│
└─ price_history table
   ├─ id (INTEGER, PRIMARY KEY)
   ├─ ticker (TEXT, FOREIGN KEY → stocks.ticker)
   ├─ price (REAL)
   ├─ recorded_at (TIMESTAMP)
   └─ Composite INDEX on (ticker, recorded_at) for fast range queries

USAGE
=====

1. Start the backend:
   $ cd backend
   $ pip install -r requirements.txt  (if not already done)
   $ python -m uvicorn main:app --reload

2. Check scraper status (watch for logs):
   $ curl http://localhost:8000/scraper/status | jq

3. Get all stocks:
   $ curl http://localhost:8000/api/stocks | jq '.data | length'

4. Get specific stock:
   $ curl http://localhost:8000/api/stocks/SCOM | jq '.data | {name, price}'

5. Monitor logs for scraper runs:
   Logs will show "Scrape job triggered" every 15 minutes between 09:00-15:00 EAT

  5. Monitor logs: grep "Scrape" backend-dev.log

If you need to force a scrape manually:
  ```python
  from scraper import scrape_nse_data
  from database import batch_insert_stocks

  stocks = scrape_nse_data()
  if stocks:
      batch_insert_stocks(list(stocks.values()))
  ```
